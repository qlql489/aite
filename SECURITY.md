# Security Policy

## 支持范围

当前优先维护最新版本。

如果你发现与以下内容相关的安全问题，欢迎反馈：

- 本地敏感信息泄露
- 权限审批绕过
- 文件系统读写越权
- 命令执行边界异常
- 自动更新链路风险
- 供应商配置或密钥暴露

## 反馈方式

为了保护用户与修复窗口，请不要直接公开披露漏洞细节。

建议通过私下渠道提交以下内容：

- 问题标题与影响范围
- 复现步骤
- 触发条件
- 预期行为与实际行为
- 影响平台
- 如果方便，附上最小复现示例或日志

## 响应原则

收到安全问题后，我们会尽快：

- 确认问题是否可复现
- 评估影响范围与修复优先级
- 在修复后安排版本更新

如果问题需要更多上下文，可能会继续联系你补充环境信息。
